# WhatsApp Appointment Bot

A WhatsApp bot that automatically:

- Monitors **Group 1**  
- Detects messages containing **[Appointment]**  
- Reformats the message into a professional format  
- Sends it to **Group 2**  

Powered by **Baileys** and designed to run on **Fly.io**.
